## 🚀 Getting Started

To run this portfolio locally, follow these simple steps:

1. **Clone the repository**

    ```bash
    git clone https://github.com/Alfaturachman/alfaturachman-portfolio.git
    ```

2. **Navigate to the project directory**

    ```bash
    cd alfaturachman-portfolio
    ```

3. **Open `index.html`**
   You can open it directly in your browser or use a live server (VS Code Live Server recommended).
